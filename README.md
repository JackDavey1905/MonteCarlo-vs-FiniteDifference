# MonteCarlo-vs-FiniteDifference
This repository investigates the calculation of the price V(t, St) and the delta of a European call option with strike price K = 162.00 and maturity T = 1.70 using two numerical methods: Monte Carlo and Finite Difference Methods. The objective of this paper is to assess which method provides the most accurate estimate of the price.
